# Valency

This organization hosts the public, MIT-licensed surface area of [Valency](https://valency.io) — the agent skills and extensions that connect Claude Code, Gemini, and other AI tools to the Valency research corpus.

## Public projects

| Repo | What it is |
|---|---|
| [`valency-plugin`](https://github.com/valency-oss/valency-plugin) | Claude Code plugin: skills + manifest |
| [`valency-gemini-extension`](https://github.com/valency-oss/valency-gemini-extension) | Gemini CLI extension installer |
| [`valency-gemini-skills`](https://github.com/valency-oss/valency-gemini-skills) | Gemini CLI skills + commands |

The Valency MCP server, ingestion pipelines, corpus data, and product backend are not open source.
